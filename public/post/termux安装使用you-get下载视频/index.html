termux安装使用you-get下载视频

2018-11-10
阅读时长: 1 分钟

You-Get简单介绍

   you-get是一个基于 Python 3 的下载工具。使用 You-Get 可以很轻松的下载到网络上的视频、图片及音乐。

一、安裝

1.更新

1

pkg update && pkg upgrade

2.获取SD卡读写权限

1

termux-setup-storage

3.安装python ffmpeg##

1

pkg install python ffmpeg

4. __升级__版本 pip版本

1

python -m pip install --upgrade pip

4. 安装you-get

1

pip install you-get

5. 更新you-get

1

pip install --upgrade you-get

二、you-get使用

查看**版本 **

1

you-get -V

查看__视频信息 __
下面我测试一下西虹市首富

1

you-get -i http://m.iqiyi.com/v_19rrf2nw1g.html

解析

1

you-get -u http://m.iqiyi.com/v_19rrf2nw1g.html


**下载 **

1

you-get -o /sdcard/video/ http://m.iqiyi.com/v_19rrf2nw1g.html


you-get的功能非常的多，如果你想了解you-get的所以使用方法可以去github you-get 项目里面查看，里面有详细使用方法，这里就简单的介绍到这里了。
