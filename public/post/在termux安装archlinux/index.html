在termux安装archLinux

2018-11-09
阅读时长: 1 分钟

termux安装archLinux

一、下载

  1. git下载
1
2
3

pkg install git
cd && git clone https://github.com/sdrausty/TermuxArch
bash TermuxArch/setupTermuxArch.sh
  1. 如果太慢可以使用wget下载
1
2
3

 pkg install wget
 
wget https://sdrausty.github.io/TermuxArch/setupTermuxArch.sh

二、安装

1

bash setupTermuxArch.sh

三、启动

1

startarch

四.修改源及更新

进入arch文件夹

1

cd arch

下面安装vim编辑工具

1

pacman -S vim

修改源为清华大学

1

vim etc/pacman.d/mirrorlist

在这上面添加

1

Server = https://mirrors.tuna.tsinghua.edu.cn/archlinuxarm/aarch64/$repo

如下图

按esc 键 :wq 保存并退出

四、更新系统

1

pacman -Syyu

五、安装常用软件

1
2
3
4

pacman -S python
pacman -S openssh
pacman -S git
pacman -S vim
