Linux常用文件传输工具使用方法

2019-06-16
阅读时长: 2 分钟

   Linux系统之间传送文件，包括Linux服务器和客户端之间，Linux主机和嵌入式Linux平台之间等，以及Linux和windows之间传送文件及同步工具；我们经常在不同的系统中或者主机设备中传输文件或者同步文件，以使用scp或者rsync及syncthing

ssh登录服务器

安装

ubuntu

1

sudo apt install ssh

CentOS

1

yum install ssh

MAC

1

brew install ssh

安卓termux

1

pkg install ssh

格式 ssh 端口 服务器$whoami@ip

1

ssh -p 22 user@192.168.1.154

scp

scp使用的是ssh的传输方式如果安装了ssh就可以直接使用

使用

  1. 复制本地文件Caddyfile到远程服务器Home目录
1

scp -P 8022 Caddyfile u0_a82@192.168.1.105:~/
  1. 把远程家目录下的Desktop/hello.py拷贝到本地当前目录下的hello.py
1

scp -P 22 user@ip:Desktop/hello.py hello.py
  1. 把本地当前目录下的demo拷贝到远程家目录下的Desktop
1

scp -P port -r demo user@ip:Desktop
  1. 把远程家目录下的Desktop拷贝到本地当前目录下的demo
1

scp -P port -r user@ip:Desktop demo

rsync

  1. 本地推送模式：
    格式：rsync -选项 -本地源文件 -远程文件夹
1

rsync -avz -e "ssh -p 8022" myblog/ u0_a82@192.168.1.104:~/

3、远程抓取模式
格式：rsync -选项 -远程文件夹 -本地目录

1

rsync -avz -P -e "ssh -p 22" user01@192.168.0.108:/tmp/test ./tmp/

syncthing

Syncthing是一个开源免费的文件夹/文件同步神器，支持Android、Linux、Windows、Mac OS X等系统，可以使我们在2台任何系统任何设备之间，实现文件实时同步，很强大。而且数据很安全，不会存储在你的设备以外的其他地方。所有通信都使用TLS进行保护。所使用的加密包括完美的前向保密，以防止窃听者获得对您的数据的访问权限。很适合我们用来搭建私有同步网盘。

安装

ubuntu

1

sudo apt install syncthing

CentOS

1

yum install syncthing

MAC

1

brew install syncthing

安卓termux

1

pkg install syncthing

使用

  • 启动
1

syncthing
