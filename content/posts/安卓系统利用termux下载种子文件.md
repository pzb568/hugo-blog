# transmission

## 1. termux里安装transmission

```
pkg install transmission
```
## 2. 启动transmission
```
transmission-daemon -T
```
## 下载管理器
### 网页管理器 
transmission GUI

<http://localhost:9091/transmission/web/>

# peerflix-server

## 安卓termux
```
pkg install nodejs
```
## centos
```

yum insatll nodejs
```
## debian/ubuntu
```
apt install nodejs
```
## 安装服务
```
npm install -g peerflix-server
```
## 启动
```
peerflix-server
```
