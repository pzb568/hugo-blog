## 查看数据库版本
```
SELECT @@VERSION;
```
## 查看数据库
```
show databases;
```
## 查看用户
```
select user();
```
## 查看用户权限
```
show grants;
```
## 创建数据库
```
create database wordpress;
```
## 删除数据库
```
drop database wordpress;
```
